# CRM Técnico

Projeto independente do dashboard de pesquisa de mercado.

## Como rodar

```bash
cd /Users/danielzanette/zanette_crm
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

O banco local do CRM sera criado automaticamente como `crm.db`.
